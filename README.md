# Facundo-Montero.github.io

Portfolio orientado a QA — clientes reales + proyectos demo, con sección de testing aplicado.

🔗 [Ver sitio](https://facundo-montero.github.io)

## Stack
HTML · CSS · JavaScript vanilla — sin build tools

## Estructura
```
index.html
css/style.css
js/main.js
img/       — capturas de cada proyecto
generate-responsive-images.js  — genera variantes srcset (requiere `npm i sharp`)
```

## Secciones
- Sitios activos (clientes en producción)
- Sitios de portfolio (demos)
- Enfoque QA — test plan + bug reports de ejemplo
